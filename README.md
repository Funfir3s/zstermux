# zstermux

一个termux一键脚本，一键安装运行各种好玩的，UnblockNeteaseMusic，aria2等等。

安装脚本：
```
pkg install git

sh -c "$(curl -fsSL https://raw.githubusercontent.com/zsxwz/zstermux/master/install.sh)"  
```

执行脚本：
```
sh zs.sh
```
结束进程
```
ps

kill pid
```
